# alertness-recognition

PCカメラ映像から、人の**眠気・注意逸脱**を推定して画面に表示するデモです。
顔のランドマークから特徴量を計算し、ルールで判定します。

> **注意:** 研究・デモ用のソフトウェアです。**安全機器ではありません。**
> 運転や作業の安全管理など、人の安全に関わる用途に単体で使わないでください。

## セットアップ（Windows）

事前に必要なのは **Python 3.10–3.12 だけ**です。無ければ <https://www.python.org/downloads/>
から入れてください（インストール時に「Add python.exe to PATH」にチェック）。
あとは `setup.bat` が仮想環境・依存導入・モデル取得まで行います。

```bat
scripts\setup.bat     :: 初回はこれだけでOK
```

後始末:

```bat
scripts\clean.bat     :: ライブラリ・モデル・キャッシュを削除（コードは残す）
scripts\uninstall.bat :: メニューで「ライブラリのみ」か「丸ごと削除」を選ぶ
```

## データ収集と採点

### 1. 集める（ガイドに従うだけ）

`collect.bat` を実行すると、最初に数秒のキャリブレーション（正面・開眼）、その後
「○○の状態にしてください」という指示が順に出ます。指示どおりに目・口・頭を動かすだけで、
自動でラベル付きデータが `runs/` に貯まります（既定で3周、区切りで合図音が鳴ります）。

```bat
scripts\collect.bat                  :: ガイドに従って収録
scripts\collect.bat --subject taro   :: 被験者IDを付ける（複数人で録るとき）
```

収録中は `q` で中断、`c` でキャリブやり直し。

### 2. 採点する

`runs/` に貯めたCSVをまとめて採点します（引数なしで `runs/` の全ファイルが対象）。

```bat
scripts\report.bat                   :: 分布＋採点表を表示
scripts\report.bat --out report.txt  :: 結果をテキストに保存（UTF-8）
```

採点表には accuracy / macro-F1 / 誤警告率 / 見逃し率 / クラス別の成績 / 混同行列が出ます。
